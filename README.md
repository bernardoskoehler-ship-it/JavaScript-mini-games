# 🎮 JavaScript Mini Games

A collection of small projects built to practice and improve JavaScript skills through hands-on development.

This repository focuses on learning by building simple interactive applications and gradually improving them over time.

---

## 🚀 Purpose

* Practice JavaScript fundamentals
* Improve logic and problem-solving
* Learn DOM manipulation
* Build a consistent coding habit
* Showcase progression (v1, v2, etc.)

---

## 📂 Projects

### 🕷️ Spider Selector (v1)

A simple interactive page where users can choose between different "Spider-Man" variations and dynamically change images.

**Concepts used:**

* DOM manipulation
* Event handling (`onclick`)
* Dynamic image updates

---

## 🧠 What I'm Learning

* Variables and functions
* Conditional logic
* DOM interaction
* Code organization
* Refactoring and improvement between versions

---

## 🔄 Versioning Approach

Each project may have multiple versions:

* **v1** → basic implementation
* **v2** → improvements and refactoring
* **v3+** → more advanced features

This helps track learning progress over time.

---

## ▶️ How to Run

1. Open any project folder
2. Navigate to a version (e.g., `v1-spider-selector`)
3. Open the `index.html` file in your browser
